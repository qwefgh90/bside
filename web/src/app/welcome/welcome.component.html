Edit and coordinate the contents in the repository on an advanced workspace

BSide is Github CMS and allows you to change and contribute the contents anywhere.

Experience the integrated workspace

The workspace provide tree explorer, editors, tabs and also tools to help editing.

Coordinate the tree structure easily

The tree explorer on left side shows a hiarachy of tree and provides various actions like moving nodes to other tree with drag and drop. It makes your coordinating files more easier.

Edit your static pages

You can edit and upload new files and rename it. Because it is based on the Monaco Editor, It also supports syntax highlighting.

Save your works without any commit

The workspace provides to save works you have done on browser storage.

Work on mobile browsers

BSide supports mobile and desktop. So you can do something with Github whenever you think of an idea.
